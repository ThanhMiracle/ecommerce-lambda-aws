## 3) Fix Docker DNS (very common)


## 4) Disable IPv6 for Docker (often fixes TLS handshake timeout)

```json
  "ipv6": false,
```


## 5) If youâ€™re behind a proxy / VPN

Then:

```bash

```
